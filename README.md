# Matrix LED Animator

## Usage

`index.html` cannot be loaded directly from a local directory due to <abbr title="Cross-Origin Resource Sharing">CORS</abbr> restrictions. It has to be loaded with an http(s):// protocol scheme.

The quickest way to do this is to run a simple web server that serves static files from its working directory:

```sh
$ python3 -m http.server
```
